# Submission Guidelines

Please submit the following:
1. SQL queries used for your analysis (in .sql format)
2. Executive summary (in .pdf format)
3. Tableau Public dashboard link
4. Any supporting documentation (if applicable)


You may submit your work as:
- Git repository, or
- ZIP file, or
- shared Google Drive link

---

## Optional
You may use a Git repository to submit your work and share the repository link.
Using Git is optional and will not affect your evaluation if not used.

---

## File Structure (Recommended)
```
submission/
│
├── sql/
│   └── analysis.sql
│
├── executive_summary.pdf (include tableau public dashboard link)
│
└── README.md (optional explanation)
```

---

## Notes
- Clearly explain any assumptions you make.
- Keep your executive summary concise and business-focused.
- Focus on insights and clarity rather than excessive formatting.
